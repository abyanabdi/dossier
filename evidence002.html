CASE FILE 001 / EVIDENCE 001-B

Website Investigation

Digital analysis of the internship application website connected to the suspected scam.

Recovered Website Evidence

Evidence Type: Website Analysis Report

Evidence ID: 001-B

Website:

globalinternships-support.com/application

Investigators were asked to determine whether this website represented a legitimate organization or a fraudulent recruitment operation.

Website Investigation Report

WEBSITE FLAG 001

Recently Created Domain

The domain was created shortly before the internship campaign began. A short domain history is a warning sign when combined with other suspicious indicators.

WEBSITE FLAG 002

Missing Organization Verification

Investigators could not locate verified company records, employee profiles, or established online accounts.

WEBSITE FLAG 003

Suspicious Website Content

The website contains generic testimonials, limited contact information, and content that appears copied from other sources.

Investigator Notes

A legitimate organization should have a consistent digital identity across multiple sources. Investigators compare domain information, company records, and online history to verify authenticity.

The website evidence suggests that the platform was designed to appear legitimate while collecting personal information.

Investigator Challenge

Which combination of findings creates the strongest evidence that this website may be fraudulent?

Reveal Investigation Notes

Correct findings:

✓ Recently created domain
✓ No verified organization records
✓ Suspicious website content
